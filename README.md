# infographic-skill

Create McKinsey-style infographics using [AntV](https://infographic.antv.vision/).

This is just a fork of the skills in https://github.com/antvis/Infographic.git translated in English for easier review.

* infographic-creator has been improved to generate embeddedable infographics for [open-slide](https://open-slide.dev/).

You can now use an agent to generate beautiful slides with infographics.

Inspired by this article: [From ArcKit to McKinsey-Style Infographics in an Afternoon](https://medium.com/arckit/from-arckit-to-mckinsey-style-infographics-in-an-afternoon-dbe0584df47a)
